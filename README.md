A simple **Weather App** built with **React** that fetches real-time weather data from a public API.  
Users can search for any city and see the current weather, temperature, and other details.

## Features
- Search for any city worldwide
- Displays temperature, weather condition, humidity, and wind speed
- Fetches real-time data from a public weather API

---

## Installation & Run

1. Clone the project:
```bash
git clone <repository-url>
```
2. Go into the project folder:

   ```bash
   cd weather-app

```
```
Install dependencies:
```bash
npm install

```
Start the development server:
```bash
npm start

```
